「文华财经风格」画线工具扩充 RFC — UI Mockup

这是画线工具扩充 RFC 的 UI 草图，用于逐屏跟你确认交互与视觉。全部沿用 2026-07-02 复盘重设计 mockup 的 iOS 视觉风格 （同一套配色 / iPhone 外框 / 顶栏与图表结构），画线形状用绝对定位的 SVG 覆盖层叠在 K 线图上。 文字标注采用「先点落点 → 原位输入」流程：先在 K 线上点一下确定落点，弹出系统键盘后文字实时显示在该落点位置，收起键盘后仍可拖气泡/尾巴微调。

① 外壳 — 进入 / 退出画线模式

顶栏「结束」左边新增固定的 ✏️ 画图 图标按钮（原稿浮动铅笔圆 .drawFab 已去掉），与「结束」之间拉开明显间距防误点。点它进入画线模式后：顶栏「结束」变「退出」且 ✏️ 图标本身也去掉——画线模式下右上角只剩「退出」一个字，不再有画图图标；底部常驻两行——上行=类型行（可收起），下行=底栏。退出画线模式后，所有已画的线不可点 / 不可拖 / 不可选中；只有在画线模式内才能选中编辑某条线。
