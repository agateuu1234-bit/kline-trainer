这版要确认的点

黏滞
长按进入 / 点击退出
长按主图弹十字光标，整图冻结（不平移/缩放）；松手后保留，点一下屏幕才退出 + 收信息栏。
自适应
信息栏躲手指
十字标在中心偏右 → 栏靠左（机 A）；在中心或偏左 → 栏靠右（机 B）。临界点 = 主图水平中线。
吸附
竖线逐根吸附 · 细实线 · 震动
竖线落最近 K 线中心；每吸附到新一根 K 线触发一次短震动(haptic)，手感清晰。横线跟手指 Y 自由读价。光标中心 = 手指触摸点（取消右偏，对齐主流），细实线非虚线。左缘价标随光标。
双面板
点哪图显哪图
长按上图 = 显示 60分 那根 K 线明细；长按下图 = 显示日线明细（各自独立）。本图演示长按上图(60分)。
价位
栏顶居中实时价 + 颜色规则
栏最上一行 = 横线在价轴上的读数（yToPrice，纯纵轴价位，与哪根 K 线无关），无标签、居中大字。颜色（2026-06-30 真机后修订）：涨=红 / 跌=绿 / 平=白（**所有价格字段**：光标价·开·高·低·收·涨跌·涨跌幅·均价，基准=前一根收盘，主流同基准）；其余=黄（日期·时间·量·额）。⚠️ 例图单元格为 brainstorming 初版（开/高/低显黄），实现以 spec §4.5 为准（全价格方向上色）。
